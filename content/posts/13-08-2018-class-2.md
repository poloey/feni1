# opening project in sublime

first open sublime editor. `File > open` folder

# boilerplate / starter tag 4

* html
* head
* title
* body
* heading (h1 - h6)
* paragraph (p)
* bold / strong (b, strong)
* italic / emphasize (i, em)
* horizontal row (hr)
* break (br)
* anchor (a)
* image tag



# basic rule
Whatever we can achieve through in css, we should use css not html.

# basic html boilerplate code

~~~html
<html>
<head>
  <title>Document</title>
</head>
<body>

</body>
</html>
~~~

# html 5 starter / boilerplate code

~~~html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Document</title>
</head>
<body>

</body>
</html>
~~~

# html5 doctype declaration

~~~html
<!DOCTYPE html>
~~~

# meta tag
meta tag used for supporting information

~~~html
<meta charset="UTF-8">
~~~

# tag, element, attribute

~~~
<!-- tag -->
html
p
hr
~~~

~~~html
<!-- element -->
<p>Lorem ipsum dolor sit amet.</p>
~~~

~~~html
<p class='hello'>Lorem ipsum dolor sit amet.</p>
<!-- here class is a attribute and class value is hello -->
~~~

# relative path and absolute path

~~~html
<!-- Relative path -->
<a href="about.html"></a>
<!-- Absolute path -->
<a href="http://google.com"></a>
~~~

Absolute path always start with `http` or `https`

# anchor tag

~~~html
<a href="about.html"></a>
~~~

# opening anchor tag in different tab
by adding `target='_blank'` attribute we can open link in different tab.
~~~html
<a href='https://google.com' target="_blank">this is google link</a>
~~~

# img tag

~~~html
<img src="imagename.jpg" alt="if image not found">
~~~
