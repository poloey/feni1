# Command

`~` - tilde means home directory


* `mkdir` - make directory - to create a folder
* `touch` - to create a file
* `ls` - listing - to check available file
* `cd` - change directory
* `rm` - remove file or folder
* `cp` - copy
* `mv` - move / cut
* `pwd` - present working directory

# flag

flag always start with hyphen `-`.

* a - all
* r - recursive
* f - force

# Identifier

* `.` - current folder
* `..` - parent folder
* `*` - everything

# Usages

How to create a folder

~~~bash
mkdir <folderName>
~~~

How to open/change/go to a folder

~~~bash
cd <foldername>
~~~

how to create file

~~~bash
touch <filename>
~~~

how to know current working directory

~~~bash
pwd
~~~

how to copy a file

~~~bash
cp <location 1> <location 2>
~~~

how to move a file

~~~bash
mv <location 1> <location 2>
~~~


how to go back to parents folder

~~~bash
cd ..
~~~


how to paste content in current folder

~~~bash
cp <location 1> .
~~~

how to copy multiple contents or folder

~~~bash
cp -r <location 1> <location 2>
~~~

how to remove a file

~~~bash
rm <filename>
~~~

how to remove a folder
~~~bash
rm -r <foldername>
~~~

how to delete everything

~~~bash
rm -r *
~~~
